Sass basics

Variables

Piense en las variables como una forma de almacenar información que desea reutilizar en toda la hoja de estilos. 

            {{code}}

Nesting

Sass te permitirá anidar tus selectores CSS de forma que sigan la misma jerarquía visual de tu HTML. 

            {{code1}}

Modules

No tienes que escribir todo tu Sass en un solo archivo. Puedes dividirlo como quieras con la regla @use. Esta regla carga otro archivo Sass como un módulo, lo que significa que puedes referirte a sus variables, mixins y funciones en tu archivo Sass con un espacio de nombres basado en el nombre del archivo. 

            {{code2}}

Mixins

Un mixin te permite crear grupos de declaraciones CSS que deseas reutilizar en todo tu sitio. Ayuda a mantener Sass muy DRY. Incluso puedes pasar valores para hacer tu mixin más flexible. 

            {{code3}}

Extend/Inheritance

Usar @extend te permite compartir un conjunto de propiedades CSS de un selector a otro. Lo que hace el código es decirle a .message, .success, que se comporten igual que %message-shared. Esto significa que en cualquier lugar donde aparezca %message-shared, también lo harán .message, .success
